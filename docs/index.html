Features

Everything a manuscript needs

✍️

Markdown editor

A clean, distraction-free writing surface. Supports all standard research paper sections — abstract, methods, results, discussion.

🔊

Read aloud

Sentence-by-sentence TTS with word-level highlighting so you can follow along and catch errors your eyes skip over.

🎙️

Voice cloning

Upload 10–20 seconds of speech and clone the voice using F5-TTS — zero-shot, no training, no cloud uploads.

📡

Built-in neural voices

10 high-quality Edge TTS voices (US, UK, AU English) that work instantly — no GPU, no downloads.

🧹

Citation stripping

Numeric [1] and author–year [Smith et al., 2020] references are removed before reading so they don't interrupt the flow.

💾

Auto-save & export

Your draft is auto-saved in the browser between sessions. Export as .md any time with Ctrl+S or the Save button.

Workflow

Three steps to a proofread draft

1

Open or paste your manuscript

Open a .docx or .md file, or paste text directly into the editor. Citations and Markdown syntax are stripped automatically before reading.

2

Click where you want to start and press Play

Place the cursor anywhere in the text and press Play. Each sentence is highlighted word-by-word as it plays. Pause, resume, or jump to any passage at any time.

3

Edit and export

Fix errors directly in the editor while listening. Press Ctrl+S or click Save .md to download the finished draft.

Requirements

What you need

OSWindows 10 or 11
Python3.12
GPUNVIDIA with CUDA 12.4 — required for voice cloning only. Built-in voices work without a GPU.
Disk~3 GB for PyTorch + F5-TTS (downloaded once on first run)
Quick start

Up and running in one double-click

Double-click launch.bat. On the first run it creates a virtual environment, installs all dependencies, and opens the browser automatically. Subsequent starts take a few seconds.

# Or install manually:
py -3.12 -m venv .venv
.venv\Scripts\activate
pip install torch==2.6.0 torchaudio==2.6.0 --index-url https://download.pytorch.org/whl/cu124
pip install -r requirements.txt
python server.py
